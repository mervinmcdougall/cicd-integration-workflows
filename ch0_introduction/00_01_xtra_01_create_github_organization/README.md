# 00_01 Extra Content 01: Create a GitHub Organization for Your Course Repositories

Use these steps to create a GitHub organization so you can keep all of your course repos in one place.

## Overview

If you follow along with the content of this course, you'll be creating lots of repositories on GitHub.

A new repo is required for many of the lessons because:

- most lessons use their own codebase which varies from other lessons
- the lessons assume you are starting from a new repo

Keeping all course-related repos in an organization helps to keep things...organized! In addition, it also prevents course-related repos from getting mixed in with personal repos.

## Prerequisites

- You’re signed in to your GitHub account at `github.com`.
- (Optional) If you’ve enabled two-factor authentication (2FA/MFA), have access to your authenticator app or verification method.

## Instructions

### 1. Open the Organizations page

1. Go to `github.com` and make sure you’re on your dashboard.
2. Select your **profile icon** in the top-right corner.
3. Select **Organizations**.

### 2. Start creating a new organization

1. On the Organizations page, select **New organization**.
2. When prompted to pick a plan, choose the **Free** plan.
3. Select **Create a free organization**.

### 3. Enter organization details

1. Enter an **organization name**.
2. Enter a **contact email**.
3. When asked who the organization belongs to, select **My personal account**.

### 4. Verify and accept terms

1. Complete the account verification step (GitHub may present a puzzle or challenge).
2. Select **Verify**.
3. Review the Terms of Service, then select the checkbox to accept them.
4. Select **Next**.

### 5. Complete setup (including MFA if enabled)

1. On the welcome screen, select **Complete setup**.
2. If you have MFA enabled, enter your **two-factor authentication code** when prompted.

Once verification is complete, your organization is created and you’ll land on the organization’s home page.

## Creating a repository in the organization

### Create from the organization home page

1. From the organization home page, select the **Repositories** tab.
2. Select **New repository** to start creating the repos you’ll use for the course.
3. Complete the remaining steps to create the repo.

### Create from github.com/new

1. Browse to GitHub's [create a new repository page, github.com/new](https://github.com/new).
2. Use the **Owner** drop down to select the organization name.
3. Complete the remaining steps to create the repo.
